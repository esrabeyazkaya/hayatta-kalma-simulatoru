Karakter Tabanlı Hayatta Kalma Simülatörü
Proje Amacı: C dili ile yazılmış, tek karakterli komutlarla oynanan bir hayatta kalma simülatörü.

Kullanılan Yapılar

Değişkenler: sağlık, enerji, yemek, sığınak

Switch-Case: komut yönetimi

If-Else: durum kontrolleri

Aritmetik & Mantıksal Operatörler: enerji/sağlık/yemek hesaplamaları

Döngüler:

FOR → tehlike dalgası

DO-WHILE → şifreli kapı

Komutlar
A	Avlan
S	Sığınak Ara
R	Dinlen
E	Envanter Göster
F	Tehlike Dalgası
P	Şifreli Geçiş
X	Çıkış
